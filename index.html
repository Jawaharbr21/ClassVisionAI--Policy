Privacy Policy

Last updated: March 2026

Welcome to ClassVisionAI. This application is developed strictly for institutional monitoring and administrative purposes. ClassVisionAI uses Artificial Intelligence and Computer Vision techniques to analyze CCTV camera images within authorized premises.

Important: This application is not intended for public or personal use. Access is restricted to authorized institutional personnel only.

1. Information We Collect

1.1 User & Institutional Information

  • Institution-issued ID (mandatory for all roles)
  • Official institutional email address
  • User role (Admin / Coordinator / Faculty)
  • Institution or organization details

1.2 CCTV & Monitoring Data

  • CCTV camera image frames
  • Timestamped monitoring data
  • AI-generated activity or anomaly insights
  • Restricted-area or policy-based monitoring metadata

1.3 Technical & System Data

  • Connected CCTV device identifiers
  • Server and application logs
  • Error reports and diagnostic data

2. Access Control & Role Approval

All users must register using a valid institutional ID. The Admin role has exclusive authority to approve, reject, or manage access requests from Coordinators and Faculty.

Coordinator and Faculty accounts remain inactive until explicitly approved by an Admin. Role-based access is enforced throughout the system.

3. How We Use the Information

  • To perform AI-based CCTV monitoring
  • To assist institutional supervision and compliance
  • To generate system alerts and monitoring insights
  • To improve AI accuracy and platform performance
  • To ensure secure, role-based system access
ClassVisionAI does not sell, rent, or share monitoring data for advertising or commercial purposes.

4. AI Usage

ClassVisionAI uses Artificial Intelligence and Computer Vision algorithms to process CCTV images for institutional monitoring. The system does not perform public identification or commercial surveillance and operates only within authorized institutional boundaries.

5. Data Security

  • Secure authentication and authorization mechanisms
  • Encrypted data transmission
  • Restricted access to sensitive monitoring data
  • Admin-controlled system permissions

6. User Rights

  • Access monitoring data based on assigned role
  • Request correction of AI-generated reports
  • Request deletion of stored monitoring data (subject to admin approval)

7. Usage Restrictions

This application is intended exclusively for authorized institutional use. Any misuse, unauthorized access, or deployment outside approved premises is strictly prohibited.

8. Policy Updates

This Privacy Policy may be updated periodically to reflect system or legal changes. Updates will be communicated through the application or institutional channels.
